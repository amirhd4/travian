## Architecture decisions
_Major design choices with rationale. The "why" matters more than the "what" for future sessions._

- **Empty ISO slots + building selection (2026-07-21)**: To match reference Travian (x1000), city slots 20-38 use "خالی" (Empty) BuildingType placeholder at level=0 instead of pre-assigned types. `_create_default_buildings()` creates empty slots with "خالی" type. `UpgradeBuildingView` handles new construction by updating `building_type` FK when `building_type_id` is provided on an empty slot, with full validation (prerequisites, duplicates, Palace uniqueness, capital restrictions, tribe restrictions). `AvailableBuildingsView` excludes "خالی" type from available list. Data migration `0012_empty_slot_building_type.py` updates existing villages. Frontend (VillageCenter.jsx) already handles empty slots correctly. **Requires `daphne` package** — Django fails to start without it (ASGI server dependency in INSTALLED_APPS).
- **Prerequisites system implemented as Python dict**: `_BUILDING_PREREQUISITES` in views.py maps building names to their prerequisite requirements (24 buildings). No DB migration needed. `_check_prerequisites()` helper validates against current village building levels. Derived from TravianX's avaliable.tpl PHP conditionals.
- **VillageCenter (dorf2) — DONE (2026-07-21)**: `travian_client/src/pages/VillageCenter.jsx` rewritten with UTF-8 Persian, iso.gif for level-0 slots, building selection modal (fetches available buildings list, shows all options with prerequisites, costs, search, and Build button calling upgrade with building_type_id). NAME_TO_GID maps Persian building names → source GID numbers. Rally point (pos 39) uses g16/g16b/g16e sprites. Wall overlay uses tribe-specific g31/g32/g33. Level indicators with toggle via localStorage key `village2_levels`.
- **Wall rendering system (2026-07-21)**: Two overlapping full-size images (628x534px) at top:0, left:0. Top layer (z-index:14, `g3XTop.png`) behind buildings; Bottom layer (z-index:39, `g3XBottom.png` or `g3XbBottom.png`) in front. Wall images do NOT change between levels 1-20 — only one image per faction. CSS specificity trap: `div.village2 #village_map img` forces 120x120px — fixed via `.wall-top`/`.wall-bottom` classes with `!important`. `overflow:'visible'` on `#village_map` allows wall to extend beyond 540x448px container. Wall enlarged to 115% with -7.5% offset. Tribe-specific: ROMAN=g31, TEUTON=g32, GAUL=g33. Defense bonus: Roman=1.030^level, Teuton=1.020^level, Gaul=1.025^level. Wall images are transparent-center borders (RGBA mode).
- **Wall level visual progression via CSS filters (2026-07-21)**: Original Travian uses same wall image for all levels. User wanted visual changes per level. Implemented using CSS filters that scale with level: brightness 0.85→1.5, contrast 0.9→1.2, saturation 0.7→2.5, sepia 0.4→0 (grayish→vivid), hue-rotate 0→-15deg (warm shift), drop-shadow 1→4px blur, golden double-glow 0→15px radius. Combined filter string applied to both Top and Bottom wall layers.
- **bgVillage-rtl.jpg is CSS-ONLY background**: The background is set via `body.village2 #wrapper { background-image: url('/assets/bgs/bgVillage-rtl.jpg'); }` in CSS. Must NOT be rendered as `<img>` inside VillageCenter.jsx. Same pattern as dorf1's f3-rtl.jpg.
- **Reference building image naming convention**: Root: `g{N}-ltr.png` / `g{N}-rtl.png`. Construction: `construction/g{N}b-rtl.png`. Resource buildings (g1-g4, g12) only have LTR. g42 only has LTR. g43/g44 don't exist in reference. WoW stages: `g40_{0-10}-rtl.png`, `g40_100-rtl.png`. Tribe-specific: `g35_brewery-rtl.png`, `g35_teahouse-rtl.png`. WoW rally: `g16_ww-rtl.png`, `g16b_ww-rtl.png`, `g16e_ww-rtl.png`.
- **Building images bulk replaced with originals (2026-07-21)**: All g1-g41 building images replaced with RTL variants from reference `Travian Sources to copy\x1000\gpack\travian_Travian_4.0_41\img\g\`. Construction variants (g{N}b) also replaced. Resource buildings (g1-g4, g12) use LTR (no RTL variant exists). Added WoW stage images (g40_0 through g40_10, g40_100), tribe-specific variants (g35_brewery, g35_teahouse), and WoW rally point variants (g16_ww, g16b_ww, g16e_ww). Removed broken g43/g43b/g44/g44b (901-byte placeholders with no reference).
